== Description ==

Extension plugin for Subscribe2 - register subscribers based on the current language.

= Prerequisites =

This plugin requires 2 pre installed and activated plugins:

* Subscribe2 (at least: 10.22.1)
* WPML Multilingual CMS (at least: 3.2)

== Installation ==

From your WordPress dashboard:

1. **Visit** Plugins > Add New
2. **Search** for "Subscriber2 WPML"
3. **Activate** Subscribe2 WPML from your Plugins page
